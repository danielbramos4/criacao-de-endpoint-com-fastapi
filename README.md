# Criação de Endpoint com FastAPI 🚀

Este projeto consiste na criação e configuração de uma API RESTful utilizando o framework **FastAPI** em Python. O objetivo principal é disponibilizar recursos relacionados a restaurantes, permitindo a consulta detalhada e a filtragem de informações de maneira prática e eficiente.

---

## 📌 Funcionalidades do Projeto

- **Criação de Ambiente e Configuração**: Setup completo do ambiente Python e instalação da biblioteca FastAPI para construção da API.
- **Endpoint de Listagem de Restaurantes**: Endpoint dedicado para disponibilizar recursos e informações específicas sobre os estabelecimentos cadastrados.
- **Filtragem por Query Parameters**: Adicionado suporte a parâmetros de consulta (`query parameters`), permitindo que os usuários busquem e filtrem restaurantes especificamente pelo nome.
- **Documentação Automática e Interativa**: Utilização das ferramentas nativas do FastAPI (`/docs` e `/redoc`) para visualização e testes diretos da API através do navegador.

---

## 🛠️ Tecnologias Utilizadas

- **Python 3.13.7**: Linguagem de programação base.
- **FastAPI**: Framework moderno e de alto desempenho para construção de APIs.
- **Uvicorn**: Servidor ASGI para carregar e rodar a aplicação Python.
